# Developer Toolbox

Essential development workflow agents for Claude Code.

## Keywords

code review, security audit, debugging, debug error, TypeError, stack trace, root cause,
TDD, test driven development, write tests, jest, vitest, pytest, coverage, failing test,
commit message, git commit, staged changes, PR description,
build verification, dist output, changes not appearing, deploy,
documentation, README, API docs, architecture diagram, user guide,
orchestrate, coordinate, multi-step, complex feature, parallel agents

## Agents Included

- commit-helper
- build-verifier
- code-reviewer
- debugger
- test-runner
- orchestrator
- documentation-expert

## Rules Included

- agent-first-thinking (behavioral interrupt for agent usage)

## Quick Install

```bash
/plugin install developer-toolbox
```
